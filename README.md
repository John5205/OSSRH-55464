# privilege

#### 介绍
权限管理模块系统

#### 软件架构
框架用到的springboot 、mybatis-plus

#### 使用说明
    1、克隆/下载 git clone https://github.com/John5205/privilege.git
    2、打包 mvn clean package
    3、maven 导入

    <dependency>
    <groupId>com.github.privilege</groupId>
        <artifactId>privilege-1.0</artifactId>
        <version>1.0</version>
    </dependency>
